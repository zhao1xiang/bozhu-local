Web 前端部署包
================

说明：此包仅包含前端代码，已配置正确的 API 路径

配置详情：
  - 跳过 Splash 页面: 是

部署步骤：
----------
1. 上传到服务器
   scp web-frontend-only.zip root@your_server_ip:/tmp/

2. 解压并替换前端文件
   cd /work/local-show
   unzip /tmp/web-frontend-only.zip
   # 或手动复制
   # cp -r frontend/* /work/local-show/frontend/

3. 重启 Nginx（可选）
   systemctl restart nginx

验证：
  - 登录接口: https://local-show.microcall.com.cn/api/auth/token
  - 健康检查: https://local-show.microcall.com.cn/health

注意：此包专为 Web 部署，EXE 版本使用独立配置
