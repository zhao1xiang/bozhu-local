# 全新电脑打包指南

本指南适用于在全新的Windows电脑上从零开始打包EXE应用。

## 📋 前置要求

### 系统要求
- Windows 10/11 64位
- 至少 8GB 内存
- 至少 20GB 可用磁盘空间
- 管理员权限

## 🔧 步骤1：安装必需软件

### 1.1 安装 Git

**下载地址：** https://git-scm.com/download/win

**安装步骤：**
1. 下载 Git for Windows 安装包
2. 运行安装程序，使用默认选项
3. 验证安装：
   ```bash
   git --version
   ```

### 1.2 安装 Node.js

**下载地址：** https://nodejs.org/

**推荐版本：** LTS 版本（当前推荐 v20.x）

**安装步骤：**
1. 下载 Windows Installer (.msi)
2. 运行安装程序，使用默认选项
3. 验证安装：
   ```bash
   node --version
   npm --version
   ```

### 1.3 安装 Python

**下载地址：** https://www.python.org/downloads/

**推荐版本：** Python 3.11 或更高

**安装步骤：**
1. 下载 Windows installer (64-bit)
2. **⚠️ 重要：勾选 "Add Python to PATH"**
3. 点击 "Install Now"
4. 验证安装：
   ```bash
   python --version
   pip --version
   ```

### 1.4 安装 Rust

**下载地址：** https://www.rust-lang.org/tools/install

**安装步骤：**
1. 下载 rustup-init.exe
2. 运行安装程序
3. 选择默认安装（输入 1）
4. 等待安装完成（可能需要10-20分钟）
5. 重启命令提示符
6. 验证安装：
   ```bash
   rustc --version
   cargo --version
   ```

### 1.5 安装 Visual Studio Build Tools

**下载地址：** https://visualstudio.microsoft.com/downloads/

**安装步骤：**
1. 下载 "Build Tools for Visual Studio 2022"
2. 运行安装程序
3. 选择 "Desktop development with C++"
4. 点击安装（约 6GB，需要较长时间）

## 🔨 步骤3：安装项目依赖

### 3.1 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
cd ..
```

**常见问题：**
- 如果安装慢，使用国内镜像：
  ```bash
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```

### 3.2 安装前端依赖

```bash
cd frontend
npm install
cd ..
```

**常见问题：**
- 如果安装慢，使用国内镜像：
  ```bash
  npm config set registry https://registry.npmmirror.com
  npm install
  ```

## 🚀 步骤4：执行打包

### 方式1：完整打包（推荐首次使用）

```bash
完整打包流程-v2.1.1.bat
```

**打包时间：** 约 15-30 分钟（首次打包）

**打包步骤：**
1. [1/6] 打包后端（约 5 分钟）
2. [2/6] 构建前端（约 3 分钟）
3. [3/6] 构建 Tauri 应用（约 10-20 分钟，首次编译）
4. [4/6] 创建分发包目录
5. [5/6] 复制文件
6. [6/6] 创建启动脚本

### 方式2：快速打包（仅前端修改）

```bash
快速重新打包前端.bat
```

**打包时间：** 约 5-10 分钟

**适用场景：** 只修改了前端代码，后端没有变化

## ⚠️ 常见问题

### 问题1：Python 命令不存在

**错误信息：** `'python' 不是内部或外部命令`

**解决方法：**
1. 重新安装 Python，确保勾选 "Add Python to PATH"
2. 或者手动添加到环境变量

### 问题2：npm install 失败

**错误信息：** `ECONNRESET` 或 `ETIMEDOUT`

**解决方法：**
```bash
# 使用国内镜像
npm config set registry https://registry.npmmirror.com
npm install
```

### 问题3：Rust 编译失败

**错误信息：** `error: linker 'link.exe' not found`

**解决方法：**
1. 安装 Visual Studio Build Tools
2. 选择 "Desktop development with C++"
3. 重启命令提示符

### 问题4：内存不足

**错误信息：** `JavaScript heap out of memory`

**解决方法：**
打包脚本已自动设置内存限制：
```bash
set NODE_OPTIONS=--max-old-space-size=4096
```

如果仍然不足，可以增加到 8192。

### 问题5：Tauri 构建时间过长

**说明：** 首次构建 Tauri 需要编译 Rust 依赖，可能需要 10-20 分钟。

**解决方法：**
- 耐心等待
- 确保网络连接稳定
- 后续构建会快很多（增量编译）
