## 一、系统登录

### 1.1 登录步骤

1. 打开眼科注射预约系统.exe
2. 输入用户名和密码
3. 点击"登录"按钮

**默认管理员账号：**
- 用户名：`admin`
- 密码：`admin`

**首次登录建议：**
- 登录后立即修改默认密码
- 在系统配置中添加其他用户账号

### 1.2 退出登录

点击右上角的用户名，选择"退出登录"。

## 三、患者管理

### 3.1 查看患者列表

1. 点击左侧菜单"患者管理"
2. 患者列表显示所有患者信息：
   - 姓名、门诊号、电话
   - 诊断、药品类型
   - 眼别、视力
   - 患者类型（初诊/经治）
   - 已完成针数

### 3.2 搜索患者

在搜索框中输入：
- 患者姓名
- 门诊号
- 电话号码

点击"搜索"按钮进行查询。

### 3.3 新建患者

#### 步骤：

1. 点击"新建患者"按钮
2. 填写患者基本信息：
   - **姓名**（必填）
   - **门诊号**（必填）
   - **电话**（必填）
   - **诊断**：选择诊断类型（如：湿性AMD、DME等）
   - **药品类型**：选择注射药品

3. 填写眼部信息：
   - **左眼**：勾选表示左眼需要治疗
   - **左眼视力**：输入视力值（如：0.5）
   - **右眼**：勾选表示右眼需要治疗
   - **右眼视力**：输入视力值

4. 选择患者类型：
   - **初诊**：首次就诊的患者
   - **经治**：已在其他医院治疗过的患者
     - 如选择"经治"，需填写"已完成针数"

5. 点击"确定"保存

#### 重要提示：

- 如果门诊号或手机号已存在，系统会提示并询问是否跳转到该患者
- 经治患者的"已完成针数"会影响后续预约的针次计算

### 3.4 编辑患者

1. 在患者列表中找到要编辑的患者
2. 点击"编辑"按钮
3. 修改患者信息
4. 点击"确定"保存

### 3.5 查看患者详情

1. 点击患者姓名或"查看"按钮
2. 查看患者的详细信息和治疗历史

### 3.6 从患者列表创建预约

1. 在患者列表中找到患者
2. 点击"预约"按钮
3. 系统自动跳转到预约页面并填充患者信息

## 四、预约管理

### 4.1 查看预约列表

1. 点击左侧菜单"预约管理"
2. 预约列表显示所有预约信息：
   - 玻注日期、注药号
   - 患者姓名、门诊号
   - 药品、眼别、针次
   - 治疗周期（强化期/巩固期）
   - 医生、费别
   - 状态（已预约/已确认/已完成/已取消）

### 4.2 筛选预约

使用筛选条件：
- **日期范围**：选择开始和结束日期
- **姓名**：输入患者姓名
- **医生**：选择医生
- **注药号**：输入注药号

点击"搜索"按钮查询，点击"重置"清除筛选条件。

### 4.3 新建预约

#### 4.3.1 单次预约

1. 点击"新建预约"按钮
2. 选择患者（可搜索姓名或电话）
3. 系统自动填充：
   - 药品类型（来自患者信息）
   - 眼别（来自患者信息）
   - 针次（根据患者历史自动计算）
   - 治疗阶段（针次≤4为强化期，>4为巩固期）
   - 玻注日期（根据系统配置的玻注日自动选择最近日期）
   - 医生和费别（延续上次预约信息）

4. 可手动调整：
   - 复诊日期
   - 玻注日期
   - 针次
   - 注药号
   - 治疗阶段
   - 左眼视力、右眼视力
   - 药品、眼别、费别
   - 医生
   - 备注

5. 点击"确定预约"保存，或点击"确定预约并打印"保存后跳转到打印页面

#### 4.3.2 批量预约（法瑞西单抗）

如果患者使用"法瑞西单抗"药品，系统会显示批量生成提示：

1. 点击"启动批量生成"按钮
2. 系统自动生成4次预约：
   - 从患者最后一次预约的针数+1开始
   - 每次间隔1个月
   - 自动选择最近的玻注日
   - 自动判断治疗阶段

3. 可以手动调整每次预约的信息
4. 可以点击"添加玻注预约日期"增加更多预约
5. 可以点击"移除"删除某次预约

#### 4.3.3 针次计算规则

系统会自动计算下一针的针次：
1. 查询患者的所有预约记录，找到最大针数
2. 查询患者表中的"已完成针数"
3. 取两者的最大值 + 1 作为下一针的针次

**示例：**
- 患者A：预约记录中最大针数为3，患者表中已完成针数为2 → 下一针为第4针
- 患者B：预约记录中最大针数为2，患者表中已完成针数为5 → 下一针为第6针
- 患者C：无预约记录，患者表中已完成针数为0 → 下一针为第1针

### 4.4 编辑预约

1. 在预约列表中找到要编辑的预约
2. 点击"编辑"按钮
3. 修改预约信息
4. 点击"确定预约"保存

**注意：** 已完成的预约不能编辑。

### 4.5 取消预约

1. 在预约列表中找到要取消的预约
2. 点击"取消"按钮
3. 确认取消操作

**注意：** 已完成和已取消的预约不能再次取消。

### 4.6 打印预约单

1. 在预约列表中找到预约
2. 点击"打印"按钮
3. 跳转到打印中心页面

## 五、日常工作

### 5.1 今日工作概览

1. 点击左侧菜单"日常工作"
2. 查看今日统计：
   - 今日预约总数
   - 已完成数量
   - 待处理数量

### 5.2 今日预约列表

显示今天的所有预约，包括：
- 患者信息
- 预约时间
- 药品、眼别、针次

1. 找到要更新的预约
2. 点击"标记为已完成"按钮
3. 预约状态更新为"已完成"

### 5.4 复诊提醒

显示即将到期的复诊患者：
- 根据系统配置的"提前提醒天数"显示
- 例如：设置提前3天提醒，则显示3天内需要复诊的患者

2. 在下拉框中选择患者（可搜索姓名或电话）

## 七、系统配置

### 7.1 通用配置

#### 7.1.1 复诊提醒天数

1. 点击左侧菜单"系统配置"
2. 找到"复诊提醒提前天数"
3. 输入天数（1-30天）
4. 点击"保存"

**说明：** 设置在复诊日期前几天出现在提醒列表中。

#### 7.1.2 玻注日配置

1. 在"玻注日"选项中选择一周中的哪些天可以进行玻璃体注药
2. 可以多选（例如：周二、周四）
3. 点击"保存"

**说明：** 
- 新建预约时，系统会自动选择最近的玻注日
- 批量生成预约时，也会根据此配置选择日期

### 7.2 数据字典管理

#### 7.2.1 医生管理

1. 点击"数据字典"标签
2. 选择"医生"类别
3. 点击"添加"按钮
4. 输入医生姓名
5. 点击"保存"

**操作：**
- 编辑：修改医生姓名
- 停用：停用后不再显示在选择列表中
- 启用：重新启用已停用的医生

#### 7.2.2 药品管理

管理可选的药品类型：
- 雷珠单抗
- 阿柏西普
- 康柏西普
- 法瑞西单抗
- 其他

#### 7.2.3 费别管理

管理费用类型：
- 自费
- 医保
- 公费
- 其他

#### 7.2.4 诊断管理

管理诊断类型：
- 湿性AMD（年龄相关性黄斑变性）
- DME（糖尿病性黄斑水肿）
- RVO（视网膜静脉阻塞）
- CNV（脉络膜新生血管）
- 其他

### 7.3 用户管理（管理员功能）

1. 点击"用户管理"标签
2. 查看所有用户列表

**添加用户：**
1. 点击"添加用户"按钮
2. 输入用户名、密码
3. 选择角色（管理员/普通用户）
4. 点击"保存"

**编辑用户：**
1. 点击"编辑"按钮
2. 修改用户信息
3. 点击"保存"

**删除用户：**
1. 点击"删除"按钮
2. 确认删除操作

## 八、常见操作流程

### 8.1 新患者首次就诊流程

1. **新建患者**
   - 进入"患者管理"
   - 点击"新建患者"
   - 填写患者信息，选择"初诊"
   - 保存

2. **创建预约**
   - 在患者列表中点击"预约"按钮
   - 或进入"预约管理"，点击"新建预约"
   - 选择患者
   - 系统自动填充信息（第1针，强化期）
   - 调整日期和其他信息
   - 保存

3. **打印预约单**
   - 点击"确定预约并打印"
   - 或在预约列表中点击"打印"
   - 打印预约单

4. **完成注药**
   - 进入"日常工作"
   - 找到该预约
   - 点击"标记为已完成"

### 8.2 经治患者转入流程

1. **新建患者**
   - 进入"患者管理"
   - 点击"新建患者"
   - 填写患者信息，选择"经治"
   - 填写"已完成针数"（例如：5）
   - 保存

2. **创建预约**
   - 系统自动从第6针开始
   - 治疗阶段自动设置为"巩固期"（因为>4针）
   - 其他步骤同上

### 8.3 法瑞西单抗批量预约流程

1. **选择患者**
   - 进入"预约管理"
   - 点击"新建预约"
   - 选择使用"法瑞西单抗"的患者

2. **批量生成**
   - 系统显示批量生成提示
   - 点击"启动批量生成"
   - 系统自动生成4次预约（间隔1个月）

3. **调整和保存**
   - 检查生成的预约信息
   - 可手动调整日期、注药号等
   - 点击"确定预约"保存

### 8.4 日常工作流程

1. **查看今日预约**
   - 进入"日常工作"
   - 查看今日预约列表

2. **处理预约**
   - 患者到达后，准备注药
   - 完成注药后，点击"标记为已完成"

3. **查看复诊提醒**
   - 查看即将到期的复诊患者
   - 提前联系患者预约

## 九、注意事项

### 9.1 数据安全

- 定期备份数据库文件（`backend/database.db`）
- 不要随意删除患者和预约记录
- 妥善保管管理员账号密码

### 9.2 操作建议

- 新建患者时，务必填写准确的门诊号和电话
- 经治患者要正确填写"已完成针数"
- 预约时检查玻注日期是否正确
- 完成注药后及时更新预约状态

### 9.3 系统配置

- 首次使用前，配置好玻注日（通常为每周固定的几天）
- 根据实际情况设置复诊提醒天数
- 及时添加和维护数据字典（医生、药品、费别等）

### 9.4 打印设置

- 首次打印前，检查打印机设置
- 建议使用A4纸张
- 打印前预览，确保内容完整
