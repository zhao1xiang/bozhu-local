## 路由结构

```
/ (根路径)
├── / → Splash页面（EXE）或跳转到 /login（Web）
├── /login → 登录页面
├── /debug → 调试信息页面
└── /app → 主应用（需要登录）
    ├── /app/dashboard → 仪表盘
    ├── /app/patients → 患者管理
    ├── /app/appointments → 预约管理
    ├── /app/daily-work → 今日预约
    ├── /app/print-center → 打印中心
    └── /app/system-config → 系统配置
```
