# 诊断和药物多选功能说明

## 功能概述

将患者的诊断和药物字段从单选改为多选，支持一个患者同时有多个诊断和多个药物。

## 主要变更

### 1. 前端表单改为多选

- **诊断字段**: 改为多选下拉框，可以选择多个诊断
- **药物字段**: 改为多选下拉框，可以选择多个药物
- **移除"其他"选项**: 多选模式下不显示"其他"选项
- **保留"其他说明"字段**: 如有其他诊断或药物，可在说明字段中输入

### 2. 数据存储格式

- **存储方式**: 多个值用逗号分隔存储在同一个字段中
- **示例**: 
  - 单个值: `"湿性AMD"`
  - 多个值: `"湿性AMD,糖尿病视网膜病变,视网膜静脉阻塞"`

### 3. 数据兼容性

#### 完全兼容旧数据

- **旧数据格式**: 单个值的字符串（如 `"湿性AMD"`）
- **新数据格式**: 逗号分隔的字符串（如 `"湿性AMD,糖尿病视网膜病变"`）
- **兼容处理**: 
  - 编辑旧数据时，自动将单个值转换为数组显示
  - 保存时，将数组转换回逗号分隔的字符串
  - 列表显示时，自动分割并以标签形式展示

#### 数据转换逻辑

```typescript
// 读取时：字符串 -> 数组
diagnosis: record.diagnosis ? record.diagnosis.split(',').map(s => s.trim()) : []

// 保存时：数组 -> 字符串
diagnosis: Array.isArray(values.diagnosis) ? values.diagnosis.join(',') : values.diagnosis || ''
```

### 4. 界面展示

#### 列表页面

- 诊断和药物以标签（Tag）形式展示
- 多个值会自动换行显示
- 不同药物有不同颜色标识

#### 表单页面

- 使用 Ant Design 的 Select 组件，mode="multiple"
- 支持搜索和多选
- 最多显示标签数量自适应（maxTagCount="responsive"）
- "其他说明"字段始终显示，不再根据选择动态显示

### 5. 批量导入

#### 模板更新

- 诊断列说明: "选填，多个用逗号分隔"
- 药物列说明: "选填，多个用逗号分隔"
- 示例数据: `"湿性AMD,糖尿病视网膜病变"`

#### 导入处理

- 支持单个值: `"湿性AMD"`
- 支持多个值: `"湿性AMD,糖尿病视网膜病变"`
- 自动去除空格和空值

### 6. 筛选功能

#### 列表筛选

- 筛选逻辑改为包含匹配（includes）
- 示例: 筛选"湿性AMD"时，会匹配包含该诊断的所有患者
- 支持筛选多诊断/多药物的患者

## 使用示例

### 添加患者

1. 点击"添加患者"按钮
2. 在诊断字段中选择多个诊断（如：湿性AMD、糖尿病视网膜病变）
3. 在药物字段中选择多个药物（如：雷珠单抗、康柏西普）
4. 如有其他诊断或药物，在"其他说明"字段中输入
5. 保存后，列表中会以标签形式显示所有选择的值

### 编辑旧数据

1. 点击编辑按钮
2. 旧的单选数据会自动转换为数组显示在多选框中
3. 可以继续添加更多选项
4. 保存后自动转换回逗号分隔的字符串格式

### 批量导入

1. 下载最新的导入模板
2. 在诊断列填写: `湿性AMD,糖尿病视网膜病变`
3. 在药物列填写: `雷珠单抗,康柏西普`
4. 上传文件，系统会自动处理

## 技术细节

### 数据库字段

- 无需修改数据库结构
- 仍使用 `diagnosis` 和 `drug_type` 字符串字段
- 通过逗号分隔存储多个值

### 前端组件

```tsx
<Select 
  mode="multiple" 
  placeholder="请选择诊断（可多选）" 
  showSearch
  maxTagCount="responsive"
>
  {diagnoses.filter(d => d.value !== '其他').map(d => (
    <Select.Option key={d.id} value={d.value}>{d.label}</Select.Option>
  ))}
</Select>
```

### 显示组件

```tsx
render: (text) => {
  if (!text) return '-';
  const items = text.split(',').map((s: string) => s.trim()).filter(Boolean);
  return (
    <Space size={[0, 4]} wrap>
      {items.map((item: string, idx: number) => (
        <Tag key={idx} color="blue">{item}</Tag>
      ))}
    </Space>
  );
}
```

## 注意事项

1. **不要选择"其他"**: 多选模式下已过滤掉"其他"选项
2. **使用"其他说明"**: 如有特殊诊断或药物，请在说明字段中输入
3. **数据兼容**: 旧数据无需迁移，系统自动兼容
4. **导入格式**: 批量导入时，多个值用英文逗号分隔
