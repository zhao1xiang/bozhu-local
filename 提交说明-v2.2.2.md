# 眼科注射预约系统 v2.2.2 提交说明

## 🎯 版本概述
- **版本号：** v2.2.2
- **发布日期：** 2026-03-13
- **类型：** 功能修复和优化版本

## 🔧 主要修复内容

### 1. 局域网访问修复
- 修复局域网访问时前端路由 404 问题
- 修复 API 端口错误（8031 → 38125）
- 后端监听 0.0.0.0，支持局域网多台电脑访问
- 前端自动检测访问 IP 类型并调整 API 地址

### 2. 预约功能完善
- 修复患者新增预约时矫正视力没有带过去的问题
- 修复药物多选时批量预约功能消失的问题
- 修复预约时间段默认值，现在默认为"上午"
- 修复从患者列表点预约时批量预约提示不显示的问题
- 修复视力字段类型问题，支持字符串格式视力值

### 3. 工作台统计优化
- 重写约针率计算逻辑，更准确反映实际约针情况
- 强化期约针率：分母为该阶段完成注药患者数，分子为已约下一针患者数
- 巩固期约针率：分母为该阶段完成注药患者数，分子为已约下一针患者数

### 4. 用户体验改进
- 调整打印页面复诊电话字体大小和位置
- 添加 SPA 路由回退机制，支持所有前端路由
- 增强错误处理和重试机制

### 5. 系统稳定性
- 自动数据库迁移功能，保留所有历史数据
- 支持视力字段的字符串格式迁移
- 自动备份机制

## 📁 修改的文件

### 后端文件
- `backend/main.py` - 主程序入口
- `backend/main_static.py` - 静态文件服务和路由回退
- `backend/simple_web_server.py` - 本地版服务器，添加数据库迁移
- `backend/models/appointment.py` - 预约模型，视力字段类型修改
- `backend/models/patient.py` - 患者模型
- `backend/routers/dashboard.py` - 工作台路由，约针率计算逻辑重写
- `backend/routers/patients.py` - 患者路由

### 前端文件
- `frontend/src/api/client.ts` - API 客户端，自动 IP 检测逻辑
- `frontend/src/pages/Appointments.tsx` - 预约页面，批量预约和视力字段修复
- `frontend/src/pages/Patients.tsx` - 患者页面
- `frontend/src/pages/PrintCenter.tsx` - 打印页面，复诊电话样式调整
- `frontend/src/types/index.ts` - 类型定义

### 新增文件
- `backend/migrate_vision_to_string.py` - 视力字段迁移脚本
- `web-dist-v2.2.2/` - 服务器版打包目录
- `simple-web-package-win7-v2.2.2/` - 本地版打包目录
- 各种打包和部署脚本
