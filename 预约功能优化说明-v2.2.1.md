# 预约功能优化说明 v2.2.1

## 功能概述

本次更新优化了预约功能，主要包括：
1. 添加治疗阶段"延长治疗期"选项
2. 玻注日期增加上午/下午时间段选择
3. 所有日期选择器改为中文显示

## 详细变更

### 1. 治疗阶段新增"延长治疗期"

#### 变更内容
- 原有阶段：强化期、巩固期
- 新增阶段：延长治疗期
- 颜色标识：
  - 强化期：蓝色
  - 巩固期：绿色
  - 延长治疗期：橙色

#### 应用场景
- 单个预约添加：可选择"延长治疗期"
- 批量预约添加：可为每个预约单独选择阶段
- 预约列表：以不同颜色标签显示

### 2. 时间段选择（上午/下午）

#### 新增字段
- 字段名：`time_slot`
- 可选值：上午、下午
- 默认值：上午

#### 功能位置
- **单个预约添加**：玻注日期下方新增"时间段"选项
- **批量预约添加**：每个预约都可以选择时间段
- **预约列表**：新增"时间段"列显示

#### 数据兼容
- 旧数据自动设置为"上午"
- 编辑旧预约时，默认显示"上午"
- 提供数据迁移脚本 `migrate_add_time_slot.py`

### 3. 中文日期选择器

#### 配置说明
在 `frontend/src/main.tsx` 中配置：
```typescript
import dayjs from 'dayjs'
import 'dayjs/locale/zh-cn'

// 配置dayjs为中文
dayjs.locale('zh-cn')
```

#### 效果
- 所有日期选择器显示中文月份和星期
- 日期格式保持 YYYY-MM-DD
- 包括：
  - 玻注日期
  - 复诊日期
  - 下次复诊日期
  - 所有其他日期选择器

## 界面变更

### 单个预约表单

```
玻注日期: [日期选择器]
时间段: ○ 上午  ○ 下午

治疗周期: ○ 强化期  ○ 巩固期  ○ 延长治疗期
```

### 批量预约表单

每行包含：
- 复诊日期
- 玻注日期
- 时间段（下拉选择：上午/下午）
- 针次
- 阶段（下拉选择：强化期/巩固期/延长治疗期）

### 预约列表

新增列：
- 时间段：显示"上午"或"下午"

治疗周期列：
- 强化期：蓝色标签
- 巩固期：绿色标签
- 延长治疗期：橙色标签

## 数据库变更

### Appointment 模型

```python
class AppointmentBase(SQLModel):
    # ... 其他字段
    time_slot: Optional[str] = None  # 时间段：上午/下午
    treatment_phase: Optional[str] = None  # 治疗周期：强化期/巩固期/延长治疗期
```

### 数据迁移

运行迁移脚本：
```bash
python backend/migrate_add_time_slot.py
```

功能：
- 为所有没有时间段的预约设置默认值"上午"
- 兼容新旧数据库

## 使用示例

### 添加单个预约

1. 选择患者
2. 选择玻注日期
3. 选择时间段（上午/下午）
4. 选择治疗周期（强化期/巩固期/延长治疗期）
5. 填写其他信息
6. 保存

### 批量添加预约

1. 选择患者
2. 点击"批量添加预约"
3. 为每个预约设置：
   - 复诊日期
   - 玻注日期
   - 时间段
   - 针次
   - 阶段
4. 保存所有预约

### 查看预约列表

- 按日期排序查看
- 时间段列显示上午/下午
- 治疗周期以彩色标签显示
- 可按各种条件筛选


### 列表显示

```tsx
// 时间段列
{
  title: '时间段',
  dataIndex: 'time_slot',
  key: 'time_slot',
  render: (text) => text || '上午',
}

// 治疗周期列
{
  title: '治疗周期',
  dataIndex: 'treatment_phase',
  key: 'treatment_phase',
  render: (text) => {
    const colorMap = {
      '强化期': 'blue',
      '巩固期': 'green',
      '延长治疗期': 'orange'
    };
    return <Tag color={colorMap[text] || 'default'}>{text || '-'}</Tag>;
  },
}
```

## 注意事项

1. **时间段默认值**：新建预约时默认为"上午"
2. **旧数据兼容**：编辑旧预约时，如果没有时间段，自动设置为"上午"
3. **治疗阶段**：延长治疗期通常用于巩固期之后的维持治疗
4. **日期格式**：所有日期选择器使用中文显示，但存储格式仍为 YYYY-MM-DD
