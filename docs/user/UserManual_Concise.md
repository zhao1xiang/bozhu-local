# 玻注预约系统 - 简明用户手册

## 1. 系统登录
*   **登录地址**：访问系统首页，自动跳转至登录页。
*   **默认账号**：`admin`
*   **默认密码**：`admin`
*   **安全建议**：首次登录后，请立即在“系统配置”中修改密码。

## 2. 工作台 (Dashboard)
登录后默认进入工作台，您可以直观查看：
*   **关键指标**：总患者数、总注药次数、今日预约数、待办随访等。
*   **数据图表**：
    *   注药趋势（月度变化）
    *   医生工作量统计
    *   药品使用分布
    *   患眼分布（左/右眼）

## 3. 每日工作 (Daily Work)
处理日常核心任务，包括预约签到和随访。
*   **预约列表**：查看当日预约，进行签到、打印等操作。
*   **随访提醒**：
    *   系统自动列出即将复诊的患者（默认提前 3 天，可配置）。
    *   **记录结果**：致电患者后，点击“记录结果”，选择状态（确认复诊、已改期、无人接听等）并填写备注。
    *   **筛选**：默认仅显示“未拨打”的记录，可切换“显示全部”查看历史记录。

## 4. 患者与预约管理
*   **患者管理**：录入新患者信息，查询历史患者。
*   **预约管理**：创建新的注药预约，调整预约时间。
*   **打印中心**：集中打印指引单、视力表等医疗文书。

## 5. 系统配置
*   **注药医生管理**：添加或编辑医生列表。
*   **通用设置**：
    *   **复诊提醒提前天数**：设置随访提醒在复诊日期前几天出现（默认为 3 天）。
*   **安全设置**：
    *   **修改密码**：输入旧密码和新密码进行修改。

## 6. 退出系统
点击左侧菜单栏底部的 **“退出登录”** 按钮即可安全退出。
