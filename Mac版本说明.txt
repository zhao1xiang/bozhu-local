下一步操作
----------------------------------------

需要在Mac系统上执行以下步骤：

1. 克隆代码仓库
   git clone <仓库地址>
   cd bozhu_local

2. 赋予执行权限
   chmod +x 打包Mac版本-完整流程.sh

3. 执行打包
   ./打包Mac版本-完整流程.sh

4. 测试运行
   cd simple-web-package-mac-v2.1.8
   ./启动服务.sh

使用方法
----------------------------------------

Mac用户收到压缩包后：

1. 解压文件
2. 双击 "启动服务.sh"
3. 如提示"无法验证开发者"：
   - 右键点击 -> 打开 -> 确认打开
4. 浏览器自动打开系统
5. 使用 admin/admin123 登录

常见问题
----------------------------------------

Q: 我没有Mac电脑怎么办？
A: 可以：
   1. 借用朋友的Mac打包
   2. 使用云端Mac服务
   3. 让Mac用户使用服务器Web版本

Q: 能在虚拟机中打包吗？
A: 理论上可以，但需要macOS虚拟机，配置较复杂

Q: Intel和M1/M2有什么区别？
A: 在对应架构的Mac上打包，生成的文件只能在该架构运行
   建议在Intel Mac上打包，可通过Rosetta 2在M1/M2上运行

Q: 需要代码签名吗？
A: 不是必须的，但建议有Apple Developer账号的进行签名
   这样用户不会看到"无法验证开发者"的警告
