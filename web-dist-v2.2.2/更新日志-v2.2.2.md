# 眼科注射预约系统 v2.2.2 更新日志

**发布日期：** 2026-03-13  
**版本类型：** 服务器版

## 主要修复

### 🔧 局域网访问修复
- 修复局域网访问时前端路由 404 问题
- 修复局域网访问时 API 端口错误（8031 → 38125）
- 后端监听 0.0.0.0，支持局域网多台电脑访问
- 前端自动检测访问 IP 类型（本地/局域网/域名）

### 🔧 预约功能修复
- 修复患者新增预约时矫正视力没有带过去的问题
- 修复药物多选时批量预约功能消失的问题（现在支持多选药物）
- 修复预约时间段默认值，现在默认为"上午"
- 修复从患者列表点预约时批量预约提示不显示的问题
- 修复视力字段类型问题，支持字符串格式视力值（如"12m", "手动"等）

### 🔧 工作台修复
- 修复工作台约针率计算逻辑，现在更准确反映实际约针情况
- **强化期约针率**：分母为该阶段完成注药患者数，分子为已约下一针患者数
- **巩固期约针率**：分母为该阶段完成注药患者数，分子为已约下一针患者数

### 🔧 打印功能优化
- 调整打印页面复诊电话字体大小和位置，更加清晰易读

### 🔧 系统稳定性
- 添加 SPA 路由回退机制，支持所有前端路由
- 自动数据库迁移，保留所有历史数据
- 增强错误处理和重试机制

## 技术改进

### 📡 API 地址自动检测
- **本地访问** (127.0.0.1) → `http://127.0.0.1:8031/api`
- **局域网访问** (192.168.x.x) → `http://192.168.x.x:8031/api`
- **域名访问** (服务器版) → `/api` (相对路径，通过 Nginx 反向代理)

### 💾 数据库兼容性
- 自动检测并迁移数据库结构
- 支持视力字段的字符串格式
- 保留所有历史数据
- 自动备份机制

### 🔄 批量预约逻辑
- 支持多选药物的批量预约
- 智能检测法瑞西单抗药物（支持字符串和数组格式）
- 优化预约流程和用户体验

## 部署说明

### 环境要求
- Python 3.7+
- 推荐使用 Nginx 作为反向代理

### 快速部署
```bash
# 1. 安装依赖
cd backend
pip install -r requirements.txt

# 2. 启动服务
chmod +x start_production.sh
./start_production.sh

# 或者直接启动
python3 -m uvicorn main:app --host 0.0.0.0 --port 8031
```

### 访问地址
- 直接访问：`http://服务器IP:8031`
- Nginx 代理：配置反向代理到 `http://localhost:8031`

### 升级说明
1. 本版本完全兼容旧版本数据
2. 首次启动会自动进行数据库迁移
3. 建议升级前备份 `database.db` 文件
4. 无需重新配置，直接替换即可使用
