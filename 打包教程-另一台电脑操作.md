## 环境准备

### 系统要求
- Windows 10/11（推荐）或 Windows 7 SP1
- 64 位操作系统
- 至少 8GB 内存
- 至少 10GB 可用磁盘空间

### 需要安装的软件

#### 1. Python 3.7.9（必需）

**为什么是 3.7.9？**
- Windows 7 兼容性最好
- PyInstaller 支持完善

**下载地址：**
https://www.python.org/ftp/python/3.7.9/python-3.7.9-amd64.exe

**安装步骤：**
1. 下载安装包
2. 运行安装程序
3. ✅ 勾选 "Add Python 3.7 to PATH"
4. 选择 "Customize installation"
5. ✅ 勾选所有 Optional Features
6. 设置安装路径（建议：`D:\workpath\python3.7.9`）
7. 点击 "Install"

**验证安装：**
```cmd
python --version
# 应该显示：Python 3.7.9
```

#### 2. Node.js LTS（必需）

**下载地址：**
https://nodejs.org/

选择 LTS 版本（推荐 18.x 或 20.x）

**安装步骤：**
1. 下载安装包
2. 运行安装程序
3. 一路 "Next"（使用默认设置）
4. 等待安装完成

**验证安装：**
```cmd
node --version
# 应该显示：v18.x.x 或 v20.x.x

npm --version
# 应该显示：9.x.x 或 10.x.x
```

#### 3. Git（可选，用于获取代码）

**下载地址：**
https://git-scm.com/download/win

**安装步骤：**
1. 下载安装包
2. 运行安装程序
3. 使用默认设置即可

## 配置环境

### 1. 修改打包脚本中的 Python 路径

打开 `打包Web版本-完整流程.bat`，找到第 9-12 行：

```batch
REM 设置 Python 3.7 路径（请根据实际情况修改）
set PYTHON37_PATH=D:\workpath\python3.7.9
set PYTHON37_EXE=%PYTHON37_PATH%\python.exe
set PIP37_EXE=%PYTHON37_PATH%\Scripts\pip.exe
set PYINSTALLER37_EXE=%PYTHON37_PATH%\Scripts\pyinstaller.exe
```

**修改为你的实际安装路径**，例如：
- 如果安装在 `C:\Python37`，改为：
  ```batch
  set PYTHON37_PATH=C:\Python37
  ```
- 如果安装在 `D:\Python\Python37`，改为：
  ```batch
  set PYTHON37_PATH=D:\Python\Python37
  ```

### 2. 检查 requirements.txt

确保 `backend/requirements.txt` 包含以下内容：

```txt
fastapi==0.104.1
uvicorn[standard]==0.24.0
sqlmodel==0.0.14
python-jose[cryptography]==3.3.0
python-multipart==0.0.6
aiofiles==23.2.1
```

**注意：不要包含 passlib 和 bcrypt**（已改用内置的 hashlib）

## 常见问题

### 问题 1：找不到 Python

**错误信息：**
```
✗ 未找到 Python 3.7: D:\workpath\python3.7.9\python.exe
```

**解决方案：**
1. 确认 Python 3.7.9 已安装
2. 修改 `打包Web版本-完整流程.bat` 中的路径
3. 重新运行打包脚本

### 问题 2：找不到 Node.js

**错误信息：**
```
✗ 未找到 Node.js
```

**解决方案：**
1. 安装 Node.js LTS 版本
2. 重启命令提示符
3. 重新运行打包脚本

### 问题 3：前端构建失败

**错误信息：**
```
✗ 前端构建失败
```

**解决方案：**
1. 删除 `frontend/node_modules` 文件夹
2. 删除 `frontend/package-lock.json` 文件
3. 重新运行打包脚本

### 问题 4：后端打包失败

**错误信息：**
```
✗ 后端打包失败
```

**解决方案：**
1. 检查 `backend/requirements.txt` 是否正确
2. 手动安装依赖：
   ```cmd
   cd backend
   D:\workpath\python3.7.9\python.exe -m pip install -r requirements.txt
   ```
3. 重新运行打包脚本

### 问题 5：内存不足

**错误信息：**
```
JavaScript heap out of memory
```

**解决方案：**
打包脚本已经设置了 `NODE_OPTIONS=--max-old-space-size=4096`，如果还是不够：

1. 关闭其他程序释放内存
2. 或者修改脚本，增加内存限制：
   ```batch
   set NODE_OPTIONS=--max-old-space-size=8192
   ```

### 问题 6：PyInstaller 版本问题

**错误信息：**
```
ModuleNotFoundError: No module named 'PyInstaller'
```

**解决方案：**
```cmd
D:\workpath\python3.7.9\Scripts\pip.exe install pyinstaller==5.13.2
```

## 附录：手动打包步骤（备用）

如果自动打包脚本失败，可以手动执行：

### 1. 安装 Python 依赖
```cmd
cd backend
D:\workpath\python3.7.9\Scripts\pip.exe install -r requirements.txt
D:\workpath\python3.7.9\Scripts\pip.exe install pyinstaller==5.13.2
```

### 2. 构建前端
```cmd
cd ..\frontend
npm install
set NODE_OPTIONS=--max-old-space-size=4096
npm run build
```

### 3. 打包后端
```cmd
cd ..\backend
D:\workpath\python3.7.9\Scripts\pyinstaller.exe build_win7_compatible.spec --clean
```

### 4. 整合文件
```cmd
cd ..
mkdir simple-web-package-win7-final
copy backend\dist\backend_server.exe simple-web-package-win7-final\
xcopy frontend\dist simple-web-package-win7-final\frontend\ /E /I /Y
copy backend\database.db simple-web-package-win7-final\
```
